#  Car Price Prediction Web Application

##  Overview
A full-stack car price prediction web application built using:
- Spring Boot (Java)
- Flask (Python)
- Random Forest Machine Learning Model
- REST API Architecture

##  Features
- Predict used car prices based on vehicle details
- Real-time ML model integration
- Clean UI with Indian currency formatting
- Deployed on Render (Cloud)

##  Tech Stack
- Java (Spring Boot)
- Python (Flask)
- Scikit-learn
- Maven
- Docker
- Render Deployment

##  Live Demo
[Live Application Link](https://car-price-springboot.onrender.com)

##  Architecture
Spring Boot → REST API → Flask ML → Prediction Response
